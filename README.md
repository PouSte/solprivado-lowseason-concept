# Villa Adela — Low-Season Landing Page (Concept)

A concept landing page for **solprivado.com**, proposed by Sol Privado / Villa Adela Real Estate S.L.

## Purpose
Highlight the **low season (November–March)** and promote winter offers, with the goal of driving
**direct bookings** through our own website — reducing OTA commission and building the Sol Privado brand.

The page presents three winter angles for the same villa:
- **Golf Escape** — golf guests near the Mar Menor courses (lead product)
- **Winter Sun Monthly Stay** — overwinterers / long stays
- **Workation** — remote workers and small teams

## Status
**Concept for design review.** Single self-contained HTML file, built to match the existing
Sol Privado brochure look & feel. No build step or dependencies.

### Intentional placeholders (to be finalised with the designer)
- **Photos** — marked photo slots indicate exactly which image goes where (hero, rooftop, golf, interior, area).
- **Guest reviews & rating score** — example text, to be replaced with real Booking.com / guest reviews.
- **CTA links** — currently point to the homepage; to be deep-linked to the live booking/availability module.
